Mappy Intelligence Documentation

Everything you need to know to make the most of Mappy Intelligence

1. Data Management 2. Visualization Options 3. Editing Competencies 4. Advanced Features

1. Data Management

Importing Data

Mappy analyzes Excel (.xlsx) files where each row represents a course and its competency coverage:

  • Select Open File from the sidebar to import your Excel file
  • Use Pre-filled to access template-based frameworks for quicker setup
  • Any non-empty text in a competency cell represents coverage and serves as justification
  • Required columns: aspect (competency name), course_code (identifier)
  • Optional columns: term, year for progression visualization

Data Management

Work with your curriculum data efficiently:

  • Use spreadsheet view for bulk editing with search capability
  • Create custom competency frameworks with the framework manager
  • Maintain multiple data sheets with independent visualizations
  • Select Save File to export in Excel (.xlsx), JSON (.json), or CSV (.csv) formats

Sheet Management

Navigate between different data sheets using the sheet selector bar. Each sheet maintains its own visualization settings and selected competencies.

2. Visualization Options

Mappy offers multiple visualization methods to analyze your curriculum data:

Radar Charts

Displays competency distribution across courses with each axis representing a competency area. Features include weighted analysis with adaptive scaling and custom color selection via grid picker.

Binary Heatmaps

Shows presence/absence of competencies across all courses using a color-coded matrix with standardized Viridis colorscale. Includes scroll indicators for large assessment datasets.

Cumulative Heatmaps

Displays the progressive competency coverage across the curriculum with independent dropdown selections and adaptive bottom margins for better display.

Educational Taxonomies

Visualizes educational frameworks with precise mathematical models:

  • Miller's Pyramid with consistent slopes and wider base
  • CDIO Framework with mathematically precise proportions
  • Dreyfus Model for skill acquisition visualization
  • Krathwohl's Affective Domain taxonomy support
  • Dynamic assessment text formatting with proper line wrapping

Elective Path Visualization

Tracks competency progression over time with position-based x-axis for clear navigation and fill-between-curves styling for better readability.

Term/Year Layout

Organizes courses by academic terms and years for a chronological view of the curriculum:

  • Use one empty row in your spreadsheet to separate terms
  • Use two consecutive empty rows to indicate year boundaries
  • This provides a temporal progression view of competency development

3. Editing Competencies

Modify competency data using multiple methods:

Direct Editing

  1. Click on any cell in the heatmap visualizations to open the editing dialog
  2. Enter or modify the justification text that explains how the competency is addressed
  3. Click Save to apply your changes to the visualization

Spreadsheet View

  1. Click the Edit Data button in the sidebar
  2. Use the spreadsheet interface to make bulk changes
  3. Use the search bar to quickly find specific content
  4. Click Save Changes to apply all modifications

Framework Management

Create and customize competency frameworks through the framework manager for template-based analysis.

Remember to use Save File after editing to persist your changes to disk.

4. Advanced Features

Visualization Export

Export your visualizations for reports and presentations:

  • Use the DL All button to download all visualizations as image files
  • Generate comprehensive PDF reports with the PDF button
  • Export in CSV, XLSX, or JSON formats for further analysis

Multi-Sheet Support

Navigate between different data sheets using the sheet selector bar below the visualization toolbar. Each sheet can contain different courses or competency frameworks.

Settings & Preferences

Access the Settings panel from the status bar to customize:

  • Appearance: Theme, color palette, font size, and chart dimensions
  • Data: Export format preferences and analysis thresholds
  • Behavior: Startup options, auto-save, and default visualization type

Framework Analysis

Analyze curriculum against various educational frameworks:

  • Miller's Pyramid for clinical skill development
  • CDIO Framework for engineering education
  • Dreyfus Model for skill acquisition
  • Krathwohl's Affective Domain taxonomy
  • Professional standards (AMC, APS, EA, SDG, etc.)

Weighted Analysis

Enable weighted analysis with the Weight toggle button to factor in assessment weightings when analyzing competency coverage.
